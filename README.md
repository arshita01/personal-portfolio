# Personal Portfolio Website 🌐

A responsive and interactive personal portfolio website built using **HTML**, **CSS (SCSS)**, and **JavaScript**. Showcases projects, skills, education, and contact information with smooth animations and a modern UI.

## ✨ Features

- Typing animation on the hero section
- Smooth scroll-based animations using **AOS** (Animate On Scroll)
- Fully responsive layout across all screen sizes
- Sections: Home, About, Resume, Projects, Contact
- Project cards with zoom-on-hover effect
- Owl Carousel for slider
- Skill progress bars
- Downloadable resume link

### Run Locally

1. **Clone the repository**
```bash
   git clone https://github.com/arshita01/personal-portfolio.git
   cd personal-portfolio
```

2. **Open in browser**
```bash
   # Simply open index.html in your browser
   open index.html
```
   Or use **Live Server** in VS Code for hot reload.

### Deploy to GitHub Pages

1. Go to your repository **Settings**
2. Navigate to **Pages**
3. Under Source, select `main` branch → `/ (root)`
4. Click **Save** — your site will be live at:
   `https://arshita01.github.io/personal-portfolio`
